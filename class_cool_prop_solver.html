Detailed Description

CoolProp solver class

This class defines a solver that calls out to the open-source CoolProp property database and is partly inspired by the fluidpropsolver that was part of the first ExternalMedia release.

libraryName = "CoolProp";

Ian Bell (ian.h.nosp@m..bel.nosp@m.l@gma.nosp@m.il.c.nosp@m.om) University of Liege, Liege, Belgium

Jorrit Wronski (jowr@.nosp@m.mek..nosp@m.dtu.d.nosp@m.k) Technical University of Denmark, Kgs. Lyngby, Denmark

2012-2014
